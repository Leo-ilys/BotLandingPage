Gift Code Redemption

To redeem a gift code, please use the `/redeem` command directly in a Discord server where the bot is present.

How to redeem:

  1. Go to any Discord channel where My Bot is active.
  2. Type /redeem
  3. Follow the bot's instructions to enter your gift code.

Example:

/redeem code:MYSECRETCODE123

If you encounter any issues, please contact an administrator on our Discord server.
