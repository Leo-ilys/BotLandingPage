Gift Code Redemption

To redeem a gift code, begin by using the `/settings` command in any Discord channel where My Bot is present.

How to redeem:

  1. Go to any Discord channel where My Bot is active.
  2. Type /settings and press Enter.
  3. In the main "Settings Menu", click the "Gift Code Operations" button.
  4. From the "Gift Code Operations" menu, if you're an alliance member redeeming for the alliance, click the "Use Gift Code for Alliance" button. For general individual redemption, your bot might monitor a "Gift Code Channel" or present another direct redemption option; please follow the bot's prompts.
  5. Follow the bot's subsequent instructions (e.g., entering your code).

Path to redemption:

/settings
→ Click "Gift Code Operations"
→ Click "Use Gift Code for Alliance" (for alliances, or follow other bot prompts)

If you encounter any issues, please contact an administrator on our Discord server.
